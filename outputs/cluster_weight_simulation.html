📐 Methodology v2 — TF-IDF Semantic Matching

🟢 Sposób Bartka (Traffic Weight)

Step 1: Extract all questions + keywords already assigned to clusters from each GEO file.
Step 2: Build a TF-IDF vectorizer (unigrams + bigrams, 5000 features) on these questions. Compute centroid vector per cluster (mean TF-IDF of all its questions).
Step 3: Fetch 1000 keywords from DataForSEO keyword_suggestions/live.
Step 4: For each keyword, compute cosine similarity against all cluster centroids. Assign to the highest-scoring cluster (min threshold: 0.01).
Step 5: Sum search volume per cluster → weight = traffic / total × 100%

🟡 Advanced Weighted Score

Same TF-IDF assignment, but instead of raw volume, uses:
Score = Vol × (1 + KD/100) × (1 + Comp) × (1 + ln(1+CPC))

Amplifies clusters with harder-to-rank keywords (high KD), more commercial intent (high CPC), and stronger advertiser competition. Shows strategic value, not just traffic volume.
